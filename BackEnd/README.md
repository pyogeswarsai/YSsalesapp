# Backend - Online Sales App

## Overview
The backend of the Online Sales App is built with Java (Spring Boot). It handles user authentication, product management, and integration with MySQL for storing data.

## Features
- RESTful APIs for user authentication and product management.
- Database integration using MySQL Workbench.
- Secure payment process implementation.

## Prerequisites
- Java 17+
- Spring Tool Suite (STS IDE)
- MySQL Workbench

## How to Run
1. Import the backend project into STS IDE.
2. Right-click on the project and select Run As -> Spring Boot App.
3. Ensure the database is set up in MySQL Workbench and the connection is configured in application.properties.

## Technologies Used
- Java 17 (Spring Boot)
- MySQL Workbench
